# AI-Based Audiogram Classification

This project focuses on applying Machine Learning and Deep Learning techniques 
to classify hearing loss from audiogram data.

## Methods Used
- Random Forest
- XGBoost
- Bi-LSTM

## Tools
Python, Google Colab, Scikit-learn, TensorFlow

## Results
Achieved high classification accuracy for hearing loss detection.
